# Types

Shared, cross-feature TypeScript types go here — e.g. types mirroring the Shared Concepts in [docs/DOMAIN_MODEL.md](../../docs/DOMAIN_MODEL.md) (`Person`, `Role`, `Money`, `Term`).

Types that belong to a single feature should live in that feature's own `types.ts` under `src/features/<feature-name>/`, not here.

This folder is intentionally empty for now.
